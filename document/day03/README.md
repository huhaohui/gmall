## SKU 和 SPU

> 简单理解为 SPU 是 Java 中的类, SKU 是 Java 中的对象

* SPU：Standard Product Unit(标准产品单位), 一组具有共同属性的商品集
* SKU：Stock Keeping Unit(库存量单位), SPU商品集因具体特性不同而细分的每个商品
